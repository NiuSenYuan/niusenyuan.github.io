CSS Grid 布局完全指南

📅 2026-04-15 ✍️ 博主 ⏱ 10 分钟阅读
CSS 前端 Web

CSS Grid 是 CSS 中最强大的布局系统。与 Flexbox 不同，Grid 是二维的——它同时处理行和列，是构建复杂页面布局的最佳选择。

基础概念

网格容器与网格项

要使用 Grid 布局，只需将元素设置为 display: grid：

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
  gap: 16px;
}

这创建了一个三列的网格，每列等宽，行高自动，单元格间距 16px。

核心属性

grid-template-columns / grid-template-rows

定义网格的列和行轨道：

/* 三种写法 */
grid-template-columns: 200px 1fr 2fr;        /* 固定 + 比例 */
grid-template-columns: repeat(3, 1fr);         /* 三等分 */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); /* 响应式 */

gap

设置行列之间的间距：

gap: 20px;              /* 行列统一间距 */
gap: 20px 10px;         /* 行间距 20px, 列间距 10px */
row-gap: 20px;
column-gap: 10px;

项目定位

通过以下属性控制单个项目的放置：

.item {
  grid-column: 1 / 3;      /* 从列线 1 到列线 3（跨两列）*/
  grid-row: 1 / 2;         /* 从行线 1 到行线 2 */
  grid-area: 1 / 1 / 2 / 3; /* 简写: row-start / col-start / row-end / col-end */
}

使用 grid-template-areas

也许是最直观的布局方式——用命名区域描述页面结构：

.layout {
  display: grid;
  grid-template-areas:
    "header  header  header"
    "sidebar content content"
    "footer  footer  footer";
  grid-template-columns: 250px 1fr 1fr;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer  { grid-area: footer; }

这就是本博客使用的布局方式！清晰、直观、可维护。

对齐方式

Grid 提供了丰富的对齐选项：

/* 容器级别 */
justify-items: center;   /* 水平方向 */
align-items: center;     /* 垂直方向 */
place-items: center;     /* 简写 */

/* 单个项目 */
justify-self: end;
align-self: start;

Grid vs Flexbox

特性 Grid Flexbox
维度 二维（行+列） 一维（行或列）
适用场景 页面布局 组件内排列
经验法则：页面整体结构用 Grid，组件内部排列用 Flexbox。两者配合使用效果最佳。

总结

CSS Grid 让复杂的网页布局变得简单。掌握 grid-template-areas、fr 单位和 auto-fill/auto-fit 这几个核心概念，你就能应对绝大多数布局需求。
