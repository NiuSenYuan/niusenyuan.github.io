Markdown 用法总结

📅 2026-04-29 ✍️ 博主 ⏱ 7 分钟阅读
Markdown 教程 写作

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它以纯文本的形式编写，可以轻松转换为 HTML。因其简洁易读的语法，被广泛应用于博客写作、项目文档、论坛帖子等场景。

标题

使用 # 号标记标题，一个 # 表示一级标题，依次类推，共支持六级标题：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

另一种方式是使用 = 和 - 分别标记一级和二级标题：

一级标题
=======

二级标题
-------

段落与换行

段落之间用空行分隔。在行末添加两个或以上空格再回车可以实现换行，也可以在行末使用 <br> 标签。

文本样式

效果 语法
粗体 **粗体** 或 __粗体__
斜体 *斜体* 或 _斜体_
粗斜体 ***粗斜体***
删除线 ~~删除线~~
行内代码 `行内代码`

列表

无序列表

使用 -、* 或 + 作为标记：

- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三

有序列表

使用数字加 . 作为标记，数字顺序不影响最终渲染：

1. 第一步
2. 第二步
   1. 子步骤 2.1
   2. 子步骤 2.2
3. 第三步

任务列表

- [x] 已完成的任务
- [ ] 待完成的任务
- [ ] 另一个待办事项

链接与图片

链接

[链接文字](https://example.com)
[链接文字](https://example.com "鼠标悬停提示")

引用式链接：
[GitHub][1]

[1]: https://github.com

图片

![替代文字](图片URL)
![替代文字](图片URL "图片标题")

也可以使用 HTML 标签来控制图片大小：

<img src="image.png" width="400" alt="描述">

引用

使用 > 来表示引用，支持嵌套引用：

> 这是一条引用
>
> > 这是嵌套引用
>
> 回到第一层引用
这是一条引用
这是嵌套引用
回到第一层引用

代码块

使用三个反引号 ``` 包裹代码，并可以指定语言以获得语法高亮：

```python
def hello():
    print("Hello, World!")
```

```javascript
console.log("Hello, World!");
```

```bash
echo "Hello, World!"
```

表格

使用 | 分隔列，使用 - 分隔表头与内容。冒号用于控制对齐方向：

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容   | 内容     | 内容   |
| 数据   | 数据     | 数据   |

水平分隔线

使用三个或以上的 ---、*** 或 ___：

---

***

___

脚注

这是一个带有脚注的句子[^1]

[^1]: 这是脚注的内容。

HTML 标签

Markdown 支持直接嵌入 HTML 标签，当 Markdown 语法无法满足需求时可以灵活使用：

<details>
<summary>点击展开</summary>

隐藏的内容在这里
</details>

转义字符

当需要使用 Markdown 保留字符本身时，使用反斜杠 \ 进行转义：

\* 不是斜体 \*
\# 不是标题
\` 不是行内代码

常用工具推荐

  • Typora：所见即所得的 Markdown 编辑器，支持即时渲染
  • VS Code：通过插件提供 Markdown 编辑和预览功能
  • Obsidian：知识管理利器，支持双向链接和关系图谱
  • Markdown Here：浏览器扩展，可在网页中即时转换 Markdown
  • Pandoc：命令行工具，支持 Markdown 与多种文档格式的相互转换

扩展语法（GFM）

总结

Markdown 语法简单直观，几分钟就能上手。它的核心优势在于——用纯文本编写，却可以生成格式丰富的 HTML 文档。无论你是写博客、做笔记、维护项目文档，还是参与开源社区，掌握 Markdown 都是一项必备技能。

记住这个原则：当你专注于内容而非排版时，Markdown 就是最好的写作工具。
